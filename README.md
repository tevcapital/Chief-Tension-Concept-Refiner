# Chief-Tension-Concept-Refiner

A multi-agent system that discovers high-quality AI agent ideas.

## How it works

1. Concept Agent → generates ideas
2. Tension Agent → critiques + upgrades ideas
3. Chief Agent → selects the strongest
4. Refiner Agent → outputs final use case

## Goal

Not to build agents — but to discover which agents are worth building.

## Run

```bash
python main.py "your domain"
